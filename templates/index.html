Word Transversal

This application recursively constructs a tree of related words based off of an initial word and a list of topics. It then saves the tree as an image and displays it for the user (which can take a bit of time depending on the depth and breadth). Only the number of iterations is required, each of the other fields will use their listed default value if none is specified.
Here's how it works:

  • 1) Initialize the current word to the initial word.
  • 2) Use the DataMuse API to search for related words to the current word that fall under the specified topics.
  • 3) Now, we look at x related words, where x is the specified breadth.
  • 4) While there are still iterations remaining, go back to step 2 with each of these words as the new current word.
